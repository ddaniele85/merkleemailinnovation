- Component Integration
Copy the CSS block <style></style> from the source HTML file.
Paste the code within the email’s <head>, before the closing </head> tag.

- Visual Customisation
For advanced customisations, refer to the Email Development Team.

- Component Dimensions
Default width: 600px.
Ensure that:
the HTML width attribute and the value defined in the inline style are consistent and identical.
Always retain the px unit in inline styles.

- Tables (Layout)
Identify all "resptab" classes applied to <table> elements.
Replace them with the classes used in the target template for responsive behaviour.

<table cellpadding="0" cellspacing="0" border="0" width="600" class="resptab" role="presentation" style="width:600px;" bgcolor="#ffffff">

to be
<table cellpadding="0" cellspacing="0" border="0" width="YOURTABLESIZE" class="TABLEMOBILECLASS" role="presentation" style="width:YOURTABLESIZEpx;" bgcolor="#ffffff">

- GIF
Image width Default: 400px
Must match the component width
Image height Default: 400px
Can be adjusted
Replace "respimg" classes with those defined in the template to ensure responsive image behaviour.
