- Component Integration
Copy the CSS block <style></style> from the source HTML file.
Paste the code within the email’s <head>, before the closing </head> tag.

- Visual Customisation
For advanced customisations, refer to the Email Development Team.

- Text
Check whether the text line already has a class attribute (having two causes problems). If it does not, create one, then add the CSS class to the tag, ensuring it is written without the dot.

<td align="center" class="fly-in-left" style="font-family: HelveticaNeue-Light, Helvetica, Arial, sans-serif; font-size: 20px; line-height: 24px; color:#000000; mso-line-height-rule: exactly; font-weight: bold">
            This is the effect!
</td>

- Images
Check that the image already has a class attribute (having two causes problems), then add the CSS class to the tag, making sure it is written without the dot.
